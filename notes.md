# sonia is an interface

sonia. a clean, command-line dsl for using a computer focused on the user experience.

```bash
sonia capture "get some milk :groc:" "cakes :esp:"
sonia add "get some cakes"
sonia remember "get some cakes too"

sonia list

sonia complete 818
sonia done 818
```

```bash
sonia change "expression" "replacement"
sonia change "expression" "replacement" 3 312 10
```

```bash
sonia forget 312 (?)
```

```bash
sonia read meditations (!)
```
